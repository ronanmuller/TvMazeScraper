*TO USE THIS API

  - Change for your connection string value on appsettings.json file
  - run Update-Database from your Package Manager Console


*TO SCRAPE AND PERSIST NEW TV SHOWS FROM PUBLIC API

  -Call URI in your browser: https://yourdomain/api/store/IDENTIFIER/save
  - IDENTIFIER == integer number (ID) which identifies the show from the public Tv Maze API



*TO SCRAPE DATA FROM LOCAL DB

   -Call URI in your browser: https://yourdomain/api/value/PAGE/PAGESIZE
   -PAGE == page index (page number) 
   -PAGESIZE == number of itens per page
